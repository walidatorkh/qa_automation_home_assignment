The search box on Canada Deals Online 
Automation Script

This script automates testing for the search box on Canada Deals Online website using PyTest and Selenium WebDriver.

Instructions:

2. Installation

	2.1 Clone or download the Python script containing the functions (link to code https://github.com/walidatorkh/qa_automation_home_assignment.git).

	2.2 Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

	2.3 Install the required dependencies using pip: pip install requests pytest pip install pytest selenium

3. Execution

	3.1 open CMD and navigate to file location
	3.2 Execute pytest -v .\test_search_box.py
